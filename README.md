# RadminPlus

**RadminPlus** - это мод для игроков Radmin VPN в Майнкрафте, который значительно расширяет возможности игры по локальной сети, добавляя удобный поиск серверов, просмотр онлайна на серверах, защиту сервера и продвинутое администрирование.

## Основные возможности

* **Удобный поиск и мониторинг:** добавляет поиск по названию или IP-адресу прямо в список локальных сетей. Показывает пинг, точный онлайн серверов и список игроков внутри при наведении.
* **Registry Sync Bypass:** если на сервере установлен мод, который запрашивает игрока при заходе установить этот же мод себе, то с RadminPlus сервер не сможет ничего запросить. игрок спокойно зайдёт на сервер без каких либо проблем и лишних скачиваний.

## Безопасность и Игровые правила (Gamerules)

Мод добавляет полезные геймрулы для защиты вашего локального сервера:
* `/gamerule illegalnick false` - запрещает заходить нелегальным никам на ваш сервер.
* `/gamerule joinundername false` - запрещает заход под ником игрока на сервер, пока тот находится в сети.
* `/gamerule register true` - включает встроенную систему регистрации и авторизации на сервере.

## Админ-команды и система прав

В моде доступен полный набор команд для администрирования:
* `/whitelist` - управление белым списком.
* `/ban` - блокировка игроков.
* `/mute` - блокировка чата.
* `/op` - продвинутая **3-уровневая система прав**:
  * `/op ник 0` - отсутствие прав (обычный игрок).
  * `/op ник 1` - возможность смены гейммода (GM) у игрока.
  * `/op ник 2` - оператор без возможности банить и т.д.
  * `/op ник 3` - полный оператор сервера на уровне хоста.
